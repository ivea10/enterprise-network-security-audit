PyFlaSQL

(Python - Flask - SQLite)

PyFlaSQL uses HTML, the standard markup language for creating web pages, along with the popular Bootstrap CSS framework to create a responsive and visually appealing user interface.

On the server-side, PyFlaSQL uses Python, along with Flask, to build the backend logic and handle dynamic content generation. To store and manage data efficiently, PyFlaSQL employs SQLite, a lightweight and serverless relational database management system.

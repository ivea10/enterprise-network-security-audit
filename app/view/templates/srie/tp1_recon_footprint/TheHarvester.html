SRIE > TP1 - Reconnaissance / Footprint > TheHarvester

TheHarvester is an open-source intelligence gathering tool designed to help security professionals and ethical hackers collect information about a target, such as email addresses, domain names, subdomains, hostnames, IP addresses, and more, from public sources.

Command Used

The "theHarvester" tool is used in the command line shell. So far it takes three parameters:

{% raw %}$ theHarvester -d {domain} -l {limit} -b {source} {% endraw %}

Help

Please refer to the following information for help on how to use this tool:

  • {domain}*

    Domain to search

  • {limit}

    Limit the number of search results.

  • {source}*

    Data source to use when gathering information.

Fields marked with * are mandatory.

Command example:
{% raw %}$ theHarvester -d raphaelviera.fr -l 300 -b google{% endraw %}

SRIE > TP1 - Reconnaissance / Footprint > IP address

The key to successfully exploiting or infiltrating a remote system lies in gathering information. The initial step in the penetration testing is reconnaissance, which involves using tools to retrieve information from the target system.

Command Used

The "IP adderss" tool is used in the command line shell. It takes no parameters:

  • Get the private IP address:
    {% raw %}$ hostname -I {% endraw %}
  • Get the public IP address:
    {% raw %}$ curl -s https://raphaelviera.fr/ismin/toolbox/my_ip.php {% endraw %}

Help

Simply copy/paste the above commands in a linux shell. There are no additional parameters required for this tool.

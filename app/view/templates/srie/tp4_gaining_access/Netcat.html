SRIE > TP4 - Gaining Access> Netcat

Netcat (often abbreviated as nc) is a versatile networking utility that reads and writes data across network connections.

Command Used

The "nc" tool is used in the command line shell. So far it takes one parameter:

{% raw %}$ nc {host} 1524{% endraw %}

Help

Please refer to the following information for help on how to use this tool:

  • {host}*

    The host on which netcat try to connect

Fields marked with * are mandatory.

Command example:
{% raw %}$ nc 192.168.255.255 {% endraw %}
